# Chess

Chess research and experiments.

The first major thread is a sub-50M-parameter Chess GM project: a compact chess model trained and evaluated with enough discipline to understand whether small models can produce strong practical chess play.

Likely work areas:

- data pipeline and position/game representation
- model architecture experiments under a sub-50M parameter budget
- training runs and ablations
- evaluation against engines and tactical suites
- self-play or generated-game experiments
- W&B tracking, checkpoints, PGNs, metrics, and analysis artifacts

This project is expected to use Scryer Io notebooks for experiment execution and Scryer/Kanbaner for planning, artifacts, and decision tracking.
